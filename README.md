# TripKeep - Twój Pamiętnik Podróży 🌍📸

Aplikacja mobilna na system Android służąca do dokumentowania wspomnień z podróży oraz planowania przyszłych wypraw. Projekt realizowany w ramach kursu "Aplikacje Mobilne".

## ✨ Funkcje
- **Dziennik wspomnień:** Dodawanie zdjęć, wideo i opisów do odbytych wypraw.
- **Planner (Bucket List):** Planowanie przyszłych podróży z odliczaniem dni.
- **Multimedialność:** Obsługa galerii zdjęć, odtwarzacza wideo oraz notatek audio.
- **Personalizacja:** Profil użytkownika ze statystykami i awatarem.
- **Offline First:** Dane przechowywane lokalnie w bazie Room.

## 🛠 Technologie
- **Język:** Kotlin
- **UI:** Jetpack Compose (Material 3)
- **Architektura:** MVVM (Model-View-ViewModel)
- **Baza danych:** Room Persistence Library
- **Multimedia:** Media3/ExoPlayer (Video/Audio), Coil (Images)
- **Nawigacja:** Jetpack Compose Navigation
